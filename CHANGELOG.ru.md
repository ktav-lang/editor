# Журнал изменений

**Languages:** [English](CHANGELOG.md) · **Русский** · [简体中文](CHANGELOG.zh.md)

Все значимые изменения в поддержке редакторов для Ktav (VS Code
расширение, IntelliJ плагин, LSP-сервер, общая TextMate-грамматика)
документируются здесь. Формат:
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/); версии
следуют [Semantic Versioning](https://semver.org/) с pre-1.0
конвенцией: MINOR-бамп считается ломающим.

Один тег (`v0.X.Y`) выпускает все четыре подпроекта одновременно.
Изменения по подпроектам сгруппированы под заголовком версии.

Этот changelog трекает **релизы поддержки редакторов / IDE**, а не
изменения формата Ktav как такового — для последних см.
[`ktav-lang/spec`](https://github.com/ktav-lang/spec/blob/main/CHANGELOG.ru.md).

## Unreleased

Первичный bootstrap репозитория. Первый публикуемый релиз будет
помечен `v0.1.0` после того, как подпроекты `grammars/`, `vscode/`,
`intellij/` и `lsp/` станут feature-complete каждый по
отдельности (см. README каждого подпроекта для статуса).
